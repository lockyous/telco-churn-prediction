# Telco Customer Churn Prediction

This project builds machine learning models to predict whether telecom customers will churn.

## Dataset
Telco Customer Churn dataset (7043 rows, 21 features)

## Workflow
- Data preprocessing
- Feature engineering
- Exploratory data analysis
- Model training and evaluation

## Models
- Naive Bayes
- K-Nearest Neighbors (best model)

## Results
Naive Bayes Test Accuracy: 74.0%  
KNN Test Accuracy: 77.5% (k = 9)

Project_1.ipynb : full notebook
